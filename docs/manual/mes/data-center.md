# MES 数据中心运维

## 这个功能做什么

数据中心用于维护 MES 页面需要使用的数据来源、字段、查询、微流和同步任务。运维人员通过这里确认页面有没有可用数据、按钮动作是否有支撑、同步任务是否正常。

## 数据中心首页

![数据中心](assets/screenshots/mes/data-center-annotated.png)

- 1：左侧菜单，进入数据中心。
- 2：操作区，用于切换功能、刷新、检测连接或新建配置。
- 3：内容区，查看数据源、工作台、字段、任务和状态。

### 常用操作

1. 进入“数据中心”。
2. 查看数据源是否存在且状态正常。
3. 使用连接检测确认数据来源可用。
4. 查看实体和字段，确认页面需要的字段已经准备好。
5. 查看查询视图或数据集，确认页面能查到数据。
6. 查看同步任务，确认最近执行状态。

### 成功标志

- 数据源连接检测成功。
- 页面需要的字段能在字段列表中看到。
- 查询结果能返回数据或明确显示空数据。
- 同步任务有正常执行记录。

## 微流管理

![微流管理](assets/screenshots/mes/data-center-microflows-annotated.png)

- 1：左侧菜单，进入微流相关功能。
- 2：操作区，用于查询、刷新、新建或执行检查。
- 3：内容区，查看微流名称、状态和最近处理结果。

### 常用操作

1. 进入数据中心中的微流管理。
2. 查询目标微流。
3. 查看状态和最近执行记录。
4. 页面按钮无响应时，先确认对应微流是否存在并启用。

### 遇到问题怎么办

- 连接检测失败：检查数据源名称、地址、账号和网络状态；处理不了时升级给系统支持人员。
- 页面字段显示不对：确认字段名称、字段类型和页面配置是否一致。
- 查询不到数据：先确认是否真的没有业务数据，再检查查询条件是否过窄。
- 微流执行失败：记录微流名称、执行时间、页面截图和操作步骤。
- 同步任务未运行：检查任务是否启用、最近执行时间和失败提示。

## 字典、编码和查询

### 用途

- 字典与编码：维护下拉选项、状态值和编号规则。
- 查询视图与数据集：给页面列表、选择器和报表提供查询结果。
- 数据同步与集成任务：让不同系统的数据按规则同步。

### 遇到问题怎么办

- 下拉选项少：检查字典项是否启用。
- 编号不连续：先确认是否允许跳号，再检查编号规则是否被修改。
- 查询结果太少：清空筛选条件后重新查询。
- 同步结果不对：查看最近一次同步时间和结果，必要时暂停后升级处理。
