# MES 应用端快速开始

## 这个功能做什么

MES 应用端用于制造执行系统的日常维护。这里可以查看首页、进入应用控制台、维护页面和数据、处理审批、管理应用内账号和运行页面。

## 谁会用

- MES 运维人员：维护页面、菜单、数据来源和发布。
- 业务管理员：查看首页、审批、公告、用户和基础资料。
- 审批管理员：维护流程、表单、审批配置和审批记录。

## 进入 MES 应用端

1. 打开管理员提供的 MES 登录页。
2. 输入有 MES 使用权限的账号和密码。
3. 点击“进入应用”。
4. 进入后确认左侧菜单包含首页、应用控制台、开发中心、数据中心、审批流和系统设置。

## 成功标志

- 登录后能看到 MES 首页。
- 左侧菜单能正常展开。
- 点击菜单后页面能显示列表、看板、表单或工作区。

## 遇到问题怎么办

### MES 登录失败

- 现象：提示账号或密码错误，或无法进入应用。
- 先检查：账号是否正确，密码是否输入完整，是否选错登录入口。
- 处理步骤：重新输入；仍失败时联系管理员确认账号是否有 MES 使用权限。
- 升级信息：提供账号、登录时间和登录页截图。

### 登录后页面空白

- 现象：能登录，但页面没有内容或一直加载。
- 先检查：刷新页面，确认网络正常。
- 处理步骤：退出后重新登录；仍空白时记录截图和操作时间。
- 升级信息：提供账号、浏览器名称、页面截图和出现时间。

### 首页数据为空

- 现象：首页指标或最近动态没有数据。
- 可能原因：当前没有业务数据，或当前账号没有查看范围。
- 处理步骤：先确认其他用户是否也为空，再检查角色和数据范围。
- 升级信息：提供账号、首页截图和期望看到的数据范围。

### 快捷入口打不开

- 现象：点击快捷入口无响应或提示无权限。
- 处理步骤：从左侧菜单直接进入同名功能；如果左侧也没有，联系管理员补充授权。
