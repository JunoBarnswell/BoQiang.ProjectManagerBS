# MES 审批、系统设置和运行页面运维

## 这个功能做什么

本章用于处理 MES 内的审批流程、应用内系统设置、正式页面和预览页面。运维人员可以查看审批进度、维护流程配置、处理用户权限，并确认运行页面是否正常。

## 审批流

![流程设计](assets/screenshots/mes/workflow-models-annotated.png)

- 1：左侧菜单，进入审批流相关功能。
- 2：操作区，用于查询、新建、编辑、启用或刷新。
- 3：内容区，查看流程、表单、配置或统计结果。

### 常用功能

- 发起申请：业务人员提交审批。
- 待办审批：审批人处理当前待办。
- 已办审批：查看自己处理过的审批。
- 我发起的：查看自己提交的审批。
- 抄送我的：查看抄送给自己的审批。
- 草稿箱：查看未提交的申请。
- 审批记录：查看审批过程和处理意见。
- 表单管理：维护审批表单。
- 流程设计：维护审批流程。
- 审批配置：把业务页面和审批流程关联起来。
- 流程分类、流程监控、审批统计、效率分析、业务数据、审批委托、消息通知设置、节假日和工作日历：用于审批运维和统计。

## 待办和审批记录

![待办审批](assets/screenshots/mes/workflow-tasks-annotated.png)

- 1：左侧菜单，进入待办审批。
- 2：操作区，用于切换待办、已办、我发起的等页签。
- 3：内容区，查看待处理事项和审批操作。

![审批记录](assets/screenshots/mes/workflow-history-annotated.png)

- 1：左侧菜单，进入审批记录。
- 2：操作区，用于查询和筛选。
- 3：内容区，查看审批过程、处理人和结果。

### 遇到问题怎么办

- 发起不了审批：确认页面是否绑定流程，表单必填项是否已填写。
- 待办看不到：确认当前用户是否是审批人，是否切到了正确页签。
- 审批按钮不可点：确认审批状态是否仍可处理，当前账号是否有处理权限。
- 审批记录缺失：按发起人、时间和标题重新筛选。
- 流程配置不生效：确认配置已保存并启用，必要时重新发布相关页面。

## MES 系统设置

![MES 用户管理](assets/screenshots/mes/system-users-annotated.png)

- 1：左侧菜单，进入 MES 系统设置。
- 2：操作区，用于查询、新增、保存、启用或停用。
- 3：内容区，查看用户、角色、菜单、公告、日志、任务、文件和打印信息。

### 常用功能

- 用户、部门、岗位：维护 MES 应用内人员和组织。
- 菜单、角色：控制用户能看到哪些功能、能点哪些按钮。
- 字典、参数、公告：维护应用内基础选项、开关和通知。
- 操作日志、登录日志、在线用户：排查用户操作和登录问题。
- 任务调度、文件中心、打印中心：查看任务、文件和打印状态。

### 遇到问题怎么办

- 用户无法进入 MES：检查账号是否启用、是否分配了 MES 角色。
- 用户看不到菜单：检查角色和菜单授权。
- 公告不显示：检查公告状态、有效时间和发布范围。
- 任务失败：查看最近执行时间和失败提示。
- 文件或打印失败：检查文件状态、模板状态和账号权限。

## 正式页面和预览页面

![运行页面](assets/screenshots/mes/runtime-order-demo-annotated.png)

- 1：左侧菜单，进入正式页面或预览页面。
- 2：操作区，用于查询、保存、执行按钮或刷新。
- 3：内容区，查看业务数据、表单内容或错误提示。

### 运维判断

- 预览页面用于发布前检查，页面内容可能随草稿变化。
- 正式页面用于业务人员日常使用，发布成功后才会更新。
- 如果预览正常但正式页面异常，先确认是否已经发布。
- 如果正式页面提示系统内部错误，记录截图和操作步骤，不要反复提交。

### 遇到问题怎么办

- 页面打不开：先刷新，再确认是否有菜单权限。
- 按钮没反应：确认按钮是否配置了动作，当前账号是否有操作权限。
- 保存失败：检查必填项、重复名称和页面提示。
- 正式页面报错：截图留存，记录页面名称、操作时间和点击步骤，交给系统支持人员。
