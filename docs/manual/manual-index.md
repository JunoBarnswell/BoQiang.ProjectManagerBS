# AsterERP 运维快速使用与故障处理手册

## 手册用途

这本手册给日常运维人员和业务管理员使用。读者不需要了解系统内部实现，只需要按图找到入口、按步骤完成操作，并在出现问题时知道先检查什么、如何处理、什么时候升级给系统支持人员。

## 使用方法

1. 先看“快速开始”，确认自己要进入平台端还是 MES 应用端。
2. 日常配置类问题看“平台运维”或“MES 运维”。
3. 新建或发布 MES 页面看“低代码页面发布”。
4. 出现错误提示、无权限、看不到菜单、发布失败、审批异常时，先看“故障处理附录”。
5. 涉及停用、删除、卸载、发布等高风险操作时，先做截图留存，并确认有可回退方案。

## 截图编号说明

所有截图上的编号含义保持一致：

- 1：左侧菜单或当前入口，用于确认自己在正确的系统位置。
- 2：操作区，通常包含查询、新增、保存、刷新、发布、切换页签等按钮。
- 3：结果区，用于查看列表、表单内容、状态提示、空数据或错误信息。

## 快速开始

### 平台端

平台端用于管理租户、应用、账号、组织、角色、公告、日志、任务、文件和打印。平台端更适合系统管理员使用。

常用动作：

1. 打开平台登录页。
2. 输入账号和密码。
3. 选择平台工作区。
4. 进入应用中心或系统设置。
5. 完成查询、维护、授权或日志查看。
6. 操作完成后退出登录，避免他人误用账号。

### MES 应用端

MES 应用端用于制造执行系统的日常维护、页面发布、数据维护、审批流和应用内账号权限。MES 应用端更适合 MES 管理员和业务运维人员使用。

常用动作：

1. 打开 MES 登录页。
2. 输入有 MES 权限的账号和密码。
3. 进入 MES 首页。
4. 根据任务进入应用控制台、开发中心、数据中心、审批流或系统设置。
5. 完成查看、配置、预览、发布或审批处理。

## 平台运维功能索引

平台管理：

- 租户管理：维护客户或组织的独立使用空间。
- 应用中心：查看应用、进入应用后台、处理发布和发布记录。
- 租户应用：给租户安装、启用、停用或卸载应用。
- 用户租户关系：把用户加入指定租户。
- 用户应用角色：给用户分配某个租户、某个应用内的角色。

系统设置：

- 用户管理：创建账号、停用账号、重置账号状态。
- 部门管理：维护组织架构。
- 岗位管理：维护岗位信息。
- 菜单管理：检查菜单是否存在和是否显示。
- 角色管理：维护角色和授权范围。
- 字典管理：维护下拉选项和状态值。
- 系统参数：维护系统运行用的开关和默认值。
- 通知公告：发布系统通知。
- 操作日志：查询用户做过哪些操作。
- 登录日志：查询用户登录记录。
- 在线用户：查看当前在线人员，必要时让账号下线。
- 任务调度：查看定时任务状态。
- 基础设施设置：查看系统基础配置项。
- 文件中心：查看上传文件和文件状态。
- 打印中心：查看打印模板和打印任务。

智能中心和扩展工具：

- AI 工作台、能力中心、模型配置、模型供应商、运行观测、安全治理、设置中心。
- Flowise 的 Chatflows、Workflows、Executions、Assistants、Marketplaces、Tools、Credentials、Variables、密钥管理、Document Stores、Evaluations、User & Workspace Management、Logs、Account Settings。
- AsterScene 的 Explore、Creator Dashboard、Assets、Pricing、Admin。
- 站内信：查看消息列表、会话、发送和接收状态。

审批流：

- 发起申请、待办审批、已办审批、我发起的、抄送我的、草稿箱、审批记录。
- 表单管理、流程设计、审批配置、流程分类、流程监控。
- 审批统计、效率分析、业务数据、审批委托、消息通知设置、节假日和工作日历。

## MES 运维功能索引

应用入口：

- 首页：查看指标、快捷入口和最近动态。
- 应用控制台：查看当前应用的管理入口。
- 开发中心：维护菜单、页面草稿、预览和发布。
- 数据中心：维护数据来源、连接检查、数据表、字段、微流和同步任务。

审批中心：

- 发起申请、待办审批、已办审批、我发起的、抄送我的、草稿箱、审批记录。
- 表单管理、流程设计、审批配置、流程分类、流程监控。
- 审批统计、效率分析、业务数据、审批委托、消息通知设置、节假日和工作日历。

MES 系统设置：

- 用户管理、部门管理、岗位管理、菜单管理、角色管理、字典管理、系统参数、通知公告。
- 操作日志、登录日志、在线用户、任务调度、基础设施设置、文件中心、打印中心。

MES 运行页面：

- 正式页面：业务人员日常使用的页面。
- 预览页面：发布前给运维确认效果的页面。
- 当前已看到的示例页面包括库存管理、出库管理、微流订单事务示例、空白页验收页面和保存页面预览。

## 故障处理总览

遇到问题先按下面顺序处理：

1. 看页面提示：记录提示文字和截图。
2. 看自己位置：确认在平台端还是 MES 应用端。
3. 看账号权限：确认是否有菜单和按钮权限。
4. 看必填项：确认红色提示和必填项已填写。
5. 看数据状态：确认数据没有被停用、删除、重复或未发布。
6. 看日志页面：平台端可查看操作日志、登录日志、任务调度和发布记录。
7. 仍无法处理时，把截图、账号、操作时间、菜单名称、处理过的步骤交给系统支持人员。

## 高风险操作提醒

下面操作不要随意提交：

- 停用租户、停用应用、卸载应用。
- 删除用户、删除角色、删除菜单、删除页面。
- 发布页面、覆盖正式页面、删除发布结果。
- 停止任务、强制用户下线、修改系统参数。

执行前必须确认影响范围、通知相关人员，并保留当前页面截图。
