# 智能中心、Flowise、AsterScene 和站内信运维

## 这个区域做什么

这些功能用于管理智能能力、自动化流程、创作资源、消息沟通和运行观察。运维人员主要负责查看状态、确认配置是否完整、定位失败提示，并把无法处理的问题升级给对应支持人员。

## 智能中心

![AI 工作台](assets/screenshots/platform/ai-workbench-annotated.png)

- 1：左侧菜单，进入智能中心相关功能。
- 2：操作区，用于切换能力、刷新状态或进入配置。
- 3：内容区，查看能力列表、运行状态和提示信息。

### 常用操作

1. 进入“智能中心”。
2. 根据任务打开 AI 工作台、能力中心、模型配置、模型供应商、运行观测、安全治理或设置中心。
3. 查看页面是否能正常加载。
4. 出现失败时记录页面提示、时间和操作步骤。

### 遇到问题怎么办

- 页面为空：先刷新页面，再确认当前账号是否有查看权限。
- 能力不可用：检查是否有启用状态、配置是否完整。
- 运行失败：查看运行观测中的失败记录，记录失败时间和任务名称。
- 安全拦截：保留提示截图，交给安全或系统支持人员确认。

## Flowise

![Flowise Chatflows](assets/screenshots/platform/flowise-chatflows-annotated.png)

- 1：左侧菜单，进入 Flowise 的各类功能。
- 2：操作区，用于查询、刷新、新建或打开记录。
- 3：内容区，展示流程、执行记录、工具、凭据、变量、文档库和日志。

### 常用操作

1. 打开 Chatflows 或 Workflows 查看流程列表。
2. 打开 Executions 查看执行结果。
3. 打开 Tools、Credentials、Variables 检查工具、凭据和变量是否存在。
4. 打开 Logs 查看运行记录。
5. 账号或工作区问题进入 User & Workspace Management 或 Account Settings。

### 遇到问题怎么办

- 流程列表为空：确认是否选错工作区或没有查看权限。
- 执行失败：先查看 Executions 和 Logs，记录失败流程名称和时间。
- 凭据缺失：在 Credentials 中确认凭据是否存在、是否被禁用。
- 工具不可用：确认 Tools 中对应工具存在且配置完整。

## AsterScene

### 常用入口

- Explore：浏览可用内容。
- Creator Dashboard：查看创作工作台。
- Assets：查看素材。
- Pricing：查看价格或套餐信息。
- Admin：进入管理入口。

### 遇到问题怎么办

- 素材打不开：确认素材是否存在、是否有访问权限。
- 工作台为空：确认当前账号是否有创作者权限。
- 管理入口打不开：确认账号是否属于管理员。
- 价格或资源显示异常：刷新后仍异常时记录截图并升级处理。

## 站内信

### 用途

站内信用于查看消息列表、会话详情、发送状态和接收状态。

### 遇到问题怎么办

- 收不到消息：确认接收人是否正确，查看消息是否发送成功。
- 会话打不开：刷新页面后重试，仍失败时记录会话名称和截图。
- 消息状态不更新：等待一段时间后刷新，仍不更新时交给系统支持人员排查。
