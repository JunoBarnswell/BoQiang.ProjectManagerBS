# 平台端快速开始

## 这个功能做什么

平台端用于维护整个系统的基础资料和运行状态。日常运维人员可以在这里管理租户、应用、账号、组织、角色、公告、日志、任务、文件和打印。

## 谁会用

- 系统管理员：维护账号、角色、组织和系统设置。
- 应用运维人员：查看应用、处理发布、进入应用后台。
- 安全或审计人员：查看登录日志、操作日志和在线用户。

## 进入平台端

1. 打开管理员提供的平台登录页。
2. 输入账号和密码。
3. 点击登录按钮。
4. 进入工作区选择页面后，选择平台工作区。
5. 进入后先确认左侧菜单是否显示“平台管理”和“系统设置”。

## 成功标志

- 登录后能看到平台工作区。
- 左侧能看到平台管理、系统设置等菜单。
- 点击菜单后页面能正常显示列表、表单或统计内容。

## 遇到问题怎么办

### 登录失败

- 现象：提示账号或密码错误。
- 先检查：账号是否输入正确，密码是否有大小写或空格。
- 处理步骤：重新输入；仍失败时联系系统管理员重置密码或检查账号状态。
- 不要做：不要连续多次盲目重试，以免触发账号锁定策略。

### 看不到平台工作区

- 现象：登录后只能看到 MES 或其他应用，看不到平台入口。
- 可能原因：当前账号没有平台端使用权限。
- 处理步骤：联系管理员确认账号是否被加入平台工作区，并分配平台角色。
- 升级信息：提供账号、登录时间和工作区选择页截图。

### 进入后菜单很少

- 现象：登录成功，但左侧没有目标菜单。
- 可能原因：账号角色不包含该菜单权限，或菜单被停用。
- 处理步骤：让管理员检查角色授权和菜单状态。
- 升级信息：提供账号、缺少的菜单名称和当前左侧菜单截图。

### 退出后无法重新进入

- 现象：退出后再次进入显示空白或仍停留在旧页面。
- 处理步骤：关闭当前页面，重新打开登录页；必要时清理浏览器缓存后重试。
- 升级信息：提供操作时间、账号和页面截图。
